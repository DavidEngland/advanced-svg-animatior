üìã How to Use in WordPress

  1. Go to your WordPress editor (Block Editor/Gutenberg)
  2. Add a Custom HTML block by clicking the "+" button and searching for "Custom HTML"
  3. Copy any of the HTML snippets above and paste them into the Custom HTML block
  4. Preview or publish your page to see the typewriter effect in action

üìù Customization Options:

  • data-text - The text to type out
  • data-typing-speed - Speed in milliseconds between each character (lower = faster)
  • data-delete-speed - Speed of deleting characters
  • data-pause-end - Pause duration after typing completes
  • data-pause-start - Pause duration before starting to delete
  • data-show-cursor - Show/hide the cursor (true/false)
  • data-cursor-char - Custom cursor character (|, _, ‚ñà, ‚ñå, etc.)
  • data-infinite-loop - Whether to loop infinitely (true/false)
